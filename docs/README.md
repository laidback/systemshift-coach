# docs/

All systemshift.coach documentation lives here. No systemshift content in `~/Documents` — this repo is the single source of truth.

## Structure

```
docs/
├── adr/          Architecture Decision Records (ArchBot produces these)
├── brand/        Brand strategy, copy, positioning
├── business/     Business plan, pricing, legal checklist
└── README.md     This file
content/          Website copy, blog drafts, social posts
```

## Rules

- **ADRs** → `docs/adr/ADR-NNN-title.md` (produced by ArchBot)
- **Brand docs** → `docs/brand/`
- **Business docs** → `docs/business/`
- **Website content drafts** → `content/`
- **Never** store systemshift docs in `~/Documents` or agent workspaces
